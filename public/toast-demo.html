How to Integrate

Step 1: Include the CSS and JS files:

<!-- In your <head> -->
<link rel="stylesheet" href="/css/toast.css">

<!-- Before closing </body> -->
<script src="/js/toast.js"></script>

Step 2: Add the toast container:

<div class="toast-container" id="toastContainer"></div>

Step 3: Show notifications:

Toast.success('Video uploaded!');
Toast.error('Upload failed.');
Toast.info('Processing...');
Toast.warning('File too large.');

Features

  • Modular & reusable across entire website
  • Auto-dismiss with customizable duration
  • Smooth slide-in/slide-out animations
  • Multiple notification types (success, error, info, warning)
  • Mobile responsive design
  • Automatic URL parameter detection
  • Clean URLs after showing notification
  • Zero dependencies (vanilla JavaScript)
  • Dark theme support
  • Stackable notifications

Quick API Reference

// Basic methods
Toast.success('Message');
Toast.error('Message');
Toast.info('Message');
Toast.warning('Message');

// Custom duration
Toast.success('Message', 5000);

// Advanced
Toast.show('Message', 'type', duration, 'title');
Toast.clearAll();
Toast.close(toastElement);
